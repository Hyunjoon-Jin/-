# 경영 · 이적 시스템 설계

> 작성일: 2026-06-30
> 상태: 설계 v0.1 (구현 동반)
> 상위 문서: [기획서 (design.md)](design.md)

4기둥 중 **구단 경영**과 **영입(이적)**을 다룬다.
선수 가치 산정 → 연봉 → 구단 재정 → 이적 시장 AI까지 한 루프로 닫는다.
모든 공식의 계수는 **초기 기준값**이며 밸런싱으로 조정한다.

---

## 1. 화폐 단위

- 내부 저장 단위: **만원** (1 = ₩10,000). 정수로 저장.
- 표시: 한국식으로 `억`/`만` 변환 (예: 350,000만원 → "35억원").
- 1억 = 10,000만원.

---

## 2. 선수 가치 (이적료 기준)

선수의 시장 가치는 다음 곱으로 산출한다.

```
value = baseValue(CA) × ageFactor(age) × potentialFactor(CA, PA, age) × contractFactor(years)
```

### 2.1 baseValue — 능력 곡선
- `baseValue = (CA / 100)^3 × 200,000만원`
- CA는 0~200 척도(능력치 평균×10). 능력이 높을수록 **가파르게** 상승(스타 프리미엄).
- 예: CA 100 → 20억, CA 130 → 약 44억, CA 160 → 약 82억.

### 2.2 ageFactor — 나이 곡선
| 나이 | 배율 |
|---|---|
| ≤ 20 | 1.10 |
| 21~24 | 1.15 (전성기 진입, 수요 최고) |
| 25~27 | 1.00 |
| 28~30 | 0.80 |
| 31~32 | 0.55 |
| 33+ | 0.30 |

### 2.3 potentialFactor — 잠재력 프리미엄
- 어린 선수가 성장 여지(PA − CA)가 크면 가치 상승.
- `potentialFactor = 1 + clamp(PA − CA, 0, 60)/60 × youthWeight`
- `youthWeight`: 21세 이하 0.8, 22~24세 0.4, 25세+ 0.1 (나이 들수록 잠재력 의미 ↓).

### 2.4 contractFactor — 잔여 계약
잔여 계약이 짧으면 자유이적 임박 → 이적료 하락.
| 잔여 | 배율 |
|---|---|
| 1년 | 0.5 |
| 2년 | 0.75 |
| 3년+ | 1.0 |

---

## 3. 연봉 (주급)

- `weeklyWage = (CA / 100)^2 × 1,500만원 × ageWageFactor`
- `ageWageFactor`: 전성기(24~30) 1.0, 어린 선수 0.7, 노장(33+) 0.9.
- 주급 합계 × 52 = 연간 인건비.

---

## 4. 구단 재정

### 4.1 상태
```ts
interface ClubFinance {
  balance: number;          // 보유 자금 (만원)
  transferBudget: number;   // 이적 가능 예산
  reputation: number;       // 1~20, 수입 규모에 영향
}
```

### 4.2 시즌 수입
| 항목 | 산식(개념) |
|---|---|
| 중계권(TV) | 평판 기반 고정 분배 |
| 입장 수입 | 홈 경기 수 × 관중 × 평판 |
| 스폰서 | 평판 기반 시즌 정액 |
| 상금 | 리그 최종 순위별 차등 |

### 4.3 시즌 지출
| 항목 | 산식 |
|---|---|
| 인건비 | 전 선수 주급 합 × 52 |
| 시설·운영 | 평판 기반 고정(추후 시설 시스템과 연동) |

### 4.4 정산
`balance += 총수입 − 총지출`. 음수가 누적되면 재정 압박(추후 실패 조건과 연결).

---

## 5. 이적 시장 (AI)

### 5.1 흐름 (이적 창)
1. 각 AI 구단이 **약점 포지션**(라인별 전력 최저)을 식별.
2. 시장 매물(타 구단 선수) 중 해당 포지션의 **가성비** 후보 탐색.
3. 예산 내에서 **이적 제안**(이적료 ≈ 시장가 ± 협상 변동) 생성.
4. 매도 구단이 수락/거절(대체 자원·재정 상황 고려, 단순화).
5. 성사 시: 이적료 이동, 선수 소속 변경, 양 구단 예산 갱신.

### 5.2 MVP 단순화
- 협상은 1라운드(제안→수락/거절)로 축소.
- 선수 개인의 이적 의사(사기·출전시간)는 추후.
- 자유계약/임대/바이아웃은 추후.
