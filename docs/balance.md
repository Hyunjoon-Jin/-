# 밸런스 리포트 & 튜닝 노트

> 최종 갱신: 2026-07-01
> 하니스: `npm run balance --workspace @soccer-tycoon/engine` (12팀 × 15시즌 멀티시즌)

헤드리스 멀티시즌 시뮬로 게임 건강도를 수치 검증하고, 문제 발견 시 상수를 조정한 기록.

## 핵심 발견: `defaultTactic` 베스트 XI 버그

가장 큰 소득. 초기 측정에서 **현재전력(CA)↔최종순위 상관이 0.26**(일부 시즌 음수)로,
강한 스쿼드가 상위권을 전혀 보장하지 못했다.

원인: AI 전술 `defaultTactic`이 포지션별 **최적 선수가 아니라 배열 순서**로 선발을 뽑고 있었다.
갓 생성된 구단은 배열 순서 = 의도한 주전이라 단일 시즌 지표는 정상(0.76)이었지만,
여러 시즌간 이적·유스·정리로 배열이 뒤섞이면 AI가 사실상 **무작위 XI**를 내보내
전력과 결과가 분리됐다. (앱의 관전/컵/자동진행 전부 영향)

수정: `defaultTactic`이 `숙련도×1000 + CA`(부상 최후순위)로 **베스트 XI**를 구성.
→ 상관이 **0.26 → 0.87**로 급등. 전력이 결과에 제대로 반영된다.

## 현재 지표 (튜닝 후)

| 지표 | 값 | 판정 |

| 우승 다양성 | 4/15팀 (1강 지배) | ⚠ CA 스프레드가 커서(86~176) 최강팀 지배 — 리그 특성상 자연스러움 |
| 자금 중앙값 | S1 111억 → S15 280억 | ✅ 지속가능(과거 7배 폭주 → 2.5배 완만) |

## 튜닝 변경 요약

- `simulateMatch`: 능력차 진폭을 `strengthSwing`(0.5) 상수로 분리(가독성).
- `tuning.baseXg`: 하향(0.135/0.115/0.09) — 베스트 XI 반영으로 상승한 득점을 목표대로.
- `matchEffects.injuryChance`: 0.008 — 이변 변수 유지.
- `finance` 수입: 중계 = 균등분(45,000) + 평판비례(rep×48,000), 스폰서 하향.
  → 약팀 최소 보장 + 폭주 억제.

## 알려진 한계

- **재정 꼬리 편차**: 임금은 선수 CA에 비례하는데 수입은 평판 고정이라,
  스쿼드가 평판을 앞서는 일부 AI 구단이 장기 적자를 누적(최저 -422억).
  파산/임금상한 시스템이 없어 방치되지만, **앱은 AI 재정을 표시하지 않아 인게임 영향 없음**.
  플레이어 구단은 능동적 관리(선수 판매·임금 통제)로 지속가능. 후속: 임금 구조/파산 도입 시 해소.
- **우승 다양성**: 테스트 리그의 CA 스프레드(91)가 현실 리그보다 크다. 1강 지배는 그 결과이며
  버그가 아님. 원한다면 tier 생성 폭을 좁혀 경쟁 균형을 높일 수 있음.
